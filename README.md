# TinyRenderer

一个纯Java语言实现的简易光栅渲染器，用于加载和渲染单个3D模型。

## 项目简介

TinyRenderer是一个轻量级的3D渲染器，完全使用Java语言实现。该项目展示了计算机图形学中的基本渲染技术，包括从模型加载到最终图像生成的全过程。

## 主要特性

- ✅ **模型加载** - 支持加载单个3D模型文件
- ✅ **纹理映射** - 实现基本的纹理贴图功能
- ✅ **阴影映射** - 支持阴影生成和渲染
- ✅ **隐藏面消除** - 实现背面剔除等优化技术
- ✅ **纯Java实现** - 不依赖外部图形库，便于学习和理解

## 项目结构

TinyRenderer/
- ├──src/ # 源代码目录
- ├──lib/ # 依赖库文件
- ├──TinyRenderer.iml # 项目配置文件
- ├──.gitignore # Git忽略文件

## 技术实现

该项目实现了以下核心图形学算法：

- 光栅化管线
- 顶点变换和投影
- 三角形填充算法
- Phong反射
- 纹理采样和映射
- 阴影计算


### 运行步骤
1. 克隆项目到本地
2. 在obj文件夹下添加需要渲染的模型
3. 在MainThread.java文件中，向models数组添加新的Model对象
4. 运行MainThread.java文件
